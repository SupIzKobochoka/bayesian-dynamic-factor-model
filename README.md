# bayes_model_latent_variable
Байесовская модель со скрытой переменной.

Все делается на основе макроэкономических данных, в качестве Y для каждой страны берутся потребление, ВВП, инвестиции.
Суть в том, что мы раскладываем Y на сумму (с коэффициентами) 3 скрытых переменных с учетом иерархии, каждая переменная оценивается через AR процесс.
Т.е. одна скрытая переменная общая у всех стран, вторая общая у всех стран одного региона, третья общая в каждой стране в отдельности.
И оценив эти переменные мы сможем получить экономическую динамику на уровнях: мировом, региональном, страновом.
